# Jellyfin

A small Android app built in a single day as a school project.

It browses popular movies and shows their details, pulling all data from [The Movie Database (TMDB)](https://www.themoviedb.org/) API.

## Setup

Add your TMDB API key to `local.properties`:

```
TMDB_API_KEY=your_key_here
```

Then build and run the app.
